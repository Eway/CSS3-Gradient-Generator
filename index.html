CSS3 Webkit Gradient Generator v1.0

Welcome to the CSS3 Webkit Gradient Generator. Please be aware that in order to make use of this utility you will need a webkit browser. This will currently not work in Internet Explorer or Firefox. Once CSS3 gradients are added to these browsers, the generator will be updated.

Remember, CSS gradients are not limited to backgrounds, they can also be used in the CSS masks as well.

How To

The generator provides a graphical user interface to generate CSS3 webkit gradients such as a gradient tool you would find in Photoshop. You can define as many color swatches as you like and use the slider to define where the color starts.

Details:

Add Swatch
Click this icon to add a new color swatch to your gradient palette.
Remove Swatch
Click this icon to remove the swatch from your gradient palette.
Select Swatch
Click the swatch to select it and then use the color picker on the right to apply a color to the selected swatch.
Adjust Color Position
Use the slider associated with each color swatch to adjust the position for the color in the gradient.

Notes:

Currently, only linear gradients can be generated. I am working on adding radial gradient generation and it should be updated soon.
