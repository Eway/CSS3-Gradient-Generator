CSS3 Webkit Gradient Generator v1.0

Welcome to the CSS3 Webkit Gradient Generator. Please be aware that in order to make use of this utility you will need a webkit browser. This will currently not work in Internet Explorer or Firefox. Once CSS3 gradients are added to these browsers, the generator will be updated.

Information

The CSS3 Webkit Gradient Generator was created as showcase of the power of CSS based gradients as well as a tool for developers and designers to generate a gradient in CSS.

Webkit gradients generate an image result, meaning the result of a webkit-gradient can be used anywhere an image can be used, be it a background-image, mask,border, or list item bullet.

Notes and Features to Come

There is still more work to come and I'm working on adding some more features to the gradient generator. Below are some features to come soon.

  • Currently only linear gradients are supported, Radial Gradients coming soon.
  • Currently only hex colors are used in the generator, RGB color support is coming soon.

Add Swatch
Click this icon to add a new color swatch to your gradient palette.
Remove Swatch
Click this icon to remove the swatch from your gradient palette.
Select Swatch
Click the swatch to select it and then use the color picker on the right to apply a color to the selected swatch.
Adjust Color Position
Use the slider associated with each color swatch to adjust the position for the color in the gradient.
Adjust Gradient Direction
You can adjust the direction of the gradient using the direction controls. You have the power to use the general keyword directions (left, right, bottom, top) or to specify numeric points for the gradient start and end points.
